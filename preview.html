🚀 Hızlı Başlangıç

1. Node.js Kurulumu:
nodejs.org adresinden LTS versiyonunu indirin ve kurun
2. Proje Kurulumu:
Terminal/PowerShell'de proje klasöründe:
npm install
3. Uygulamayı Başlatın:
npm run dev
Sonra tarayıcıda: http://localhost:3000
✨ Proje hazır! Node.js kurulduktan sonra 3D drone trajectory planlama deneyimini yaşayın.
